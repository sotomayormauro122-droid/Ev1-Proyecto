# Proyecto Web Hamnet

## Descripción

Este proyecto es un sitio web sobre la película Hamnet, desarrollado con HTML, CSS y JavaScript. Incluye secciones como inicio, sinopsis, tráiler, reparto, críticas, compra de entradas y un formulario de contacto.

Se utilizaron estilos con Flexbox y algunos efectos con CSS y JavaScript para mejorar la apariencia y la interacción.

## Tecnologías

HTML5
CSS3
JavaScript
Git y GitHub
